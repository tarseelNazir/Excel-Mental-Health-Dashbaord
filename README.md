# Excel-Mental-Health-Dashbaord
This project uses Microsoft Excel to analyze mental health data and generate meaningful insights through data cleaning, Pivot Tables, Charts, and Dashboard visualizations. The analysis helps identify patterns related to stress, anxiety, work-life balance, and overall mental well-being.

## Features
- Data Cleaning and Processing
- Pivot Tables and Pivot Charts
- Interactive Dashboard
- Mental Health Trend Analysis
- Visual Representation of Survey Results

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting
- Data Analysis

## Objective
To transform raw mental health survey data into meaningful insights that help understand factors influencing mental well-being.
